bmctool.simulation.simulate

bmctool.simulation.simulate(config_file: str | Path, seq_file: str | Path, show_plot: bool = False, **kwargs) → BMCSim[source]

Run BMCTool simulation based on given seq-file and config file.

Parameters:

  • config_file – Path to the YAML config file

  • seq_file – Path to the pulseq sequence file

  • show_plot – Flag to activate plotting of simulated data, by default False

  • optional – Flag to activate plotting of simulated data, by default False

  • **kwargs – Additional keyword arguments passed to the plot_z function

Returns:

BMCTool object containing the simulation results.

Return type:

BMCTool

Raises:

FileNotFoundError – If the config_file or seq_file not found.
